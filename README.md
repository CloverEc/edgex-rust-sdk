# EdgeX Rust SDK

This is a Rust SDK for interacting with the EdgeX exchange.

> [!WARNING]
> This project is currently under active development. APIs and features are subject to change.

## Features

- Async runtime support with `tokio`
- HTTP client using `reqwest`
- WebSocket support
- Cryptographic signing (Ethereum/Starknet style)

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
edgex-rust-sdk = { path = "." }
```
